== Description ==

Commentag is a Widget that helps your visitors catch up with any discussion easily:

*	Get an overview of any discussion at a glance
*	Filter comments by tags
*	Don't replace your commenting system

== Installation ==

Please visit http://www.commentag.com to register and download our plugin!

= Why isn't the plugin on wp-plugin.com?  =
